Getting started guide

This guide walks through the three steps needed to stand up a local development environment from a clean checkout of the repository.

Install the toolchain

First install the runtime and the package manager. The full matrix of supported versions is maintained on the supported versions page.

Verify the installation

Run the doctor script to confirm every dependency resolved correctly. If a check fails, consult the troubleshooting reference before opening an issue.

Run the test suite

The fast smoke gate finishes in about thirty seconds and is the quickest way to confirm your environment is wired up end to end.
